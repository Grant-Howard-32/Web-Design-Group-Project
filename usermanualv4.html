User Manual

Adding an Instructor

To add an instructor, follow these steps:

  1. Click on the "User Plus" icon in the navigation menu
  2. Select "Add Instructor" from the dropdown menu
  3. Enter the instructor's information in the form provided
  4. Click the "Submit" button to add the instructor

Enrolling a Student

To enroll a student, follow these steps:

  1. Click on the "User Plus" icon in the navigation menu
  2. Select "Enroll Student" from the dropdown menu
  3. Enter the student's information in the form provided
  4. Click the "Submit" button to enroll the student

Adding a Course

To add a course, follow these steps:

  1. Click on the "Plus Circle" icon in the navigation menu
  2. Select "Add Course" from the dropdown menu
  3. Enter the course information in the form provided
  4. Click the "Submit" button to add the course

Registering for a Course

To register for a course, follow these steps:

  1. Click on the "Plus Circle" icon in the navigation menu
  2. Select "Register Course" from the dropdown menu
  3. Select the course you want to register for from the list
  4. Click the "Submit" button to register for the course

Dropping a Course

To drop a course, follow these steps:

  1. Click on the "Plus Circle" icon in the navigation menu
  2. Select "Drop Course" from the dropdown menu
  3. Select the course you want to drop from the list
  4. Click the "Submit" button to drop the course
